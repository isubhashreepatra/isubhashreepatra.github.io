HOW TO PLAY?

RULES: Pick a color that matches the RGB color value displayed on top. You score a point if you pick the right one in the first attempt.

INFORMATION: RGB stands for Red, Green and Blue. RGB color system constructs all the colors from the combination of the Red, Green and Blue colors.

  • Red + Green = Yellow
  • Red + Blue = Magenta
  • Green + Blue = Cyan
  • Red + Green + Blue = White
